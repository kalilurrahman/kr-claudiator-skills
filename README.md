## Usage

Place a skill directory under `/mnt/skills/` and reference it in your Claude system prompt or tool configuration.

Each SKILL.md contains:
- Frontmatter (name, description, argument-hint, allowed-tools)
- Step-by-step process
- Production-ready output templates with working code examples
- Worked examples with realistic scenarios
- Anti-patterns table
- 10 rules
